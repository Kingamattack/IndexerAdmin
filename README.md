# IndexerAdmin
iOS Admin app for Indexer Project (Objective-C)

Librairies/API:
  * Map: MapKit
  * Http: AFNetworking (https://github.com/AFNetworking/AFNetworking)
  * Local database: Realm (https://github.com/realm/realm-cocoa)
